# TestNGAutomation
## Test flow driven framework associated with TestNG and Selenium Libraries, built on Java.

### Pros
1. HTML reports
2. CLI run capable
3. Ready to build
4. Grid compatiable
5. Multibrowser support
6. Organized test data with excel
7. Parameterized

### Cons
1. Non Mavenized
2. No Error Handling
3. Basic setup
